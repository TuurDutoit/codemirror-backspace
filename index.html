CodeMirror 6: Broken backspace

Python

Type an underscore pretty much anywhere in the document. You won't be able to backspace anymore until you type something else.

Only on iOS devices; it works fine on Android and desktop.

SQL

After a valid query, add some nonsense and a quote, e.g. SELECT 'SQL'; the". You won't be able to remove it at all. By tapping backspace repeatedly, you can sometimes make the cursor move back a few characters, but it won't remove the text.

Broken on iOS, Android and desktop!
